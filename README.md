### The Atlas Six (Flagship Project)

<a href="https://github.com/Passion-Over-Pain/The-Atlas-Six" target="_blank">
<img width="100%" src="https://github.com/user-attachments/assets/714bfaf1-0fc3-4ea2-a644-c6da4d16e84b" />
</a>

Alright… this one is personal.

I built a desktop fantasy RPG without using a game engine. Yes, actually.

It started as a simple idea between friends. Just characters and random lore.  
Then I kept building… and building… until it became a full system.

Now it’s a JavaScript-powered desktop game running on Electron.

Key systems include:

• JSON-driven branching story engine  
• Recursive event-based architecture (the entire game flows through it)  
• Turn-based combat inspired by Slay the Spire  
• Inventory, skill trees, and progression  
• File-based save/load system using Node.js  

It’s not fully released yet, still actively being refined (v1.0.3),  
but it already has early traction and 23 GitHub stars.

This project taught me more than any course ever could.

🔗 https://github.com/Passion-Over-Pain/The-Atlas-Six
